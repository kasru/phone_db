# Проект phone_db — телефонная база данных на C

Библиотека для хранения и обработки телефонных номеров с комментариями и датами окончания действия. Поддерживает пакетную загрузку CSV-файлов (до 300 млн записей), инкрементальные операции (add/update/delete), поиск и сохранение в отсортированном порядке.

## Задание

См. [readme.txt](readme.txt) — полное описание задания.

## Сборка и запуск тестов

```bash
make              # сборка и запуск всех тестов
make test         # только функциональные тесты (44 шт.)
make test-units   # только unit-тесты (26 шт.)
make clean        # удаление собранных файлов
```

Требования: GCC, стандарт C11.

## Структура проекта

| Файл | Описание |
|------|----------|
| `phone_db.h` | Заголовочный файл: типы и прототипы функций |
| `phone_db.c` | Реализация всех функций |
| `test_phone_db.c` | Функциональные тесты (44 шт.) |
| `test_units.c` | Unit-тесты (26 шт.) |
| `test_framework.h` | Фреймворк тестирования |
| `Makefile` | Сборка проекта |
